## 功能列表

- 官/B服游戏启动(制作时以官服为例，不保证B服可用)

- 每日套餐青春版
> 只能一次性执行 喝咖啡（可选咖啡），报刊亭刮刮乐，录像店经营，领取日常奖励

> 录像店经营不匹配热门推荐，可以选择不让脚本经营

- 官/B服游戏关闭(区服选项在游戏启动)
